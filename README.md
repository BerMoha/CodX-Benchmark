# CodX Benchmark v1.0

A public, verifiable benchmark comparing CodX against single frontier AI models on code intelligence tasks.

## What is CodX?
CodX is a proprietary AI code intelligence platform built by Solvatex.
Architecture details are confidential — results are fully verifiable.

## Missions
1. Code generation
2. Security audit
3. System audit
4. Code review
5. Debug & fix

## How to verify
1. Download prompts from /prompts/
2. Run them on any model
3. Apply rubric from /rubric/scoring_criteria.md
4. Compare with published results in /results/
